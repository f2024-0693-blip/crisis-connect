Problem Statement
Pakistan's disaster response is unstructured and slow

Pakistan faces annual floods, earthquakes, fires, and traffic disasters with no unified AI-powered incident classification system. Citizens cannot quickly report disasters in a structured way, and emergency responders lack automated tools to prioritise incidents by type and severity. Crisis Connect addresses this gap by building a multimodal AI classifier that takes a photo and a short text description from a citizen and instantly classifies the disaster type while estimating severity level.

Key Novelty
Multimodal fusion

Most existing disaster classifiers use images alone. Crisis Connect combines image classification (ResNet-50 CNN) with text classification (TF-IDF + MLP) in a weighted fusion — improving accuracy from 91% to 95.07%. Supports audio input via Whisper speech-to-text for hands-free reporting in emergencies.
